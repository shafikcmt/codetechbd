1. Introduction
- About Mikrotik

- How to choose Mikrotik Router Model based on Business Requirements

- Winbox Accessing
- Admin Password Changing
- User Creation
- IP service list choosing and its benefits
- Default OS packages, new package install, package uninstall, upgrade & downgrade
- History Checking
- SNMP setting
- Resources checking
- Routerboard Details
- Mikrotik Command Prompt/Terminal
- System Configuration Backup Export & Import
- Mikrotik Reboot
- Mikrotik Reset

3. IP Configuration & NAT
- IP Subnet
- WAN IP Configuration
- DNS adding
- Private IP NAT
- LAN IP Configuration
- Static IP LAN

- DHCP server setup
- Leases management
- IP Pool

- MAC Bonding
- Internet Access Control

5. Port Bridging

- Several ISP Load Balancing
